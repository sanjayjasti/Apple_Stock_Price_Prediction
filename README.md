# Apple Stock Price Prediction using Deep Learning (LSTM)

## Project Overview

This project predicts the next day's Apple stock price using a Long Short-Term Memory (LSTM) neural network. Live stock market data is fetched from Yahoo Finance and processed to generate future price predictions.

The project also compares the performance of traditional Machine Learning models such as Linear Regression and Random Forest with the Deep Learning LSTM model.

## Features

* Live Apple stock data from Yahoo Finance
* LSTM-based next-day stock price prediction
* Actual vs Predicted stock price visualization
* Current stock price display
* Predicted next-day stock price
* Historical stock price analysis
* Flask-based web application
